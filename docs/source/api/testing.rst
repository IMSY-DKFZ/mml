mml.testing
===========

The testing sub-module provides utilities to test ``mml-core`` as well as plugins. The :mod:`~mml.testing.fixtures`
module is linked as a ``pytest`` entry point and thus loaded for every pytest session within a virtual environment
that has installed ``mml-core``.
