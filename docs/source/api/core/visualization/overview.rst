mml.core.visualization
======================

``MML`` gives some visualization options for tasks and image examples. Be aware that for the training visualization
(e.g. metrics) the corresponding ``exp_logger`` is responsible (default: ``tensorboard``). This sub-module is less
advanced and welcomes any contributions.


  * :doc:`cm` - visualize a confusion matrix
  * :doc:`predictions` - visualize samples images from a task
  * :doc:`logo` - ``MML`` logo for the terminal
  * :doc:`utils` - visualization utils
